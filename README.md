# site-autotesting-2

### Позитивный тест-кейс:
- **Название**: Успешное заполнение и отправка формы
- **Шаги**:
  1. Открыть страницу с формой
  2. Заполнить поле "Name" валидными данными
  3. Заполнить поле "Password" валидными данными
  4. Из списка What is your favorite drink? выбрать Milk и Coffee
  5. Из списка What is your favorite color? выбрать Yellow
  6. В поле Do you like automation? выбрать любой вариант
  7. Заполнить поле "Email" валидным email
  8. В поле Message написать количество инструментов, описанных в пункте Automation tools, и написать инструмент из списка Automation tools, содержащий наибольшее количество символов
  9. Нажать кнопку "Submit"
  10. Проверить появление всплывающего окна с сообщением об успехе
- **Ожидаемый результат**: Форма успешно отправлена, появляется сообщение "Message received!"

  8. В поле Message написать количество инструментов, описанных в пункте Automation tools, и написать инструмент из списка Automation tools, содержащий наибольшее количество символов
